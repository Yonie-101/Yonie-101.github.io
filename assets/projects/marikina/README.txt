========================================================
                 README FILE - MarikinaGo
========================================================

Project Title: Developing an Interactive Platform for 
Promoting Local Tourism and Enhancing Visitor Experience 
in Marikina City
Capstone Project - BSIT, FEU Roosevelt Marikina

--------------------------------------------------------
1. Overview
--------------------------------------------------------
MarikinaGo is a mobile application designed to enhance
tourism engagement and streamline booking experiences
for visitors of Marikina City. The app provides a
comprehensive guide to destinations, cultural sites,
restaurants, and services, while also offering booking
and navigation features to ensure a smooth user
experience.

--------------------------------------------------------
2. Installation Guide
--------------------------------------------------------
To install MarikinaGo on your Android device:

1. Copy the file "base.apk" from the memory card
   to your device.
2. Enable "Install from Unknown Sources" in Settings.
3. Locate the APK file and tap to install.
4. Once installed, open the app from your home screen.
5. Ensure internet connectivity for full feature access.

--------------------------------------------------------
3. Key Features (App)
--------------------------------------------------------
Homepage
- Looping background video for visual engagement
- Horizontal category carousel: Resto, Culture,
  Museum, Church, Hotels, Parks, Shops
- Dynamic display of destinations based on category

Browsing Places
- Curated list of destinations per category
- Detailed view with image gallery, descriptions,
  fun facts, weather tips, entrance fees, mood tags,
  operating hours, and contact information
- Embedded map and external location links
- User reviews and feedback

Explore Page
- Recommended destinations
- Search bar for places and tour packages
- Filters: sort by rating, filter by category, etc.

Mapping System
- Geolocation-based exploration
- Hamburger menu for search and category browsing
- Destination location display
- Navigation modes: Driving, Walking, Cycling Route
- Location setting: manual search, map click,
  automatic detection, virtual location

Booking Page
- View available tour packages
- Login required for booking
- Input guest details: male/female count, agency type,
  guest type, status, tour packages
- Group booking support
- Tour start and end date selection
- Checkout view with itemized inclusions and total price
- Payment processing and receipt generation (Ref ID)
- Confirmation from Marikina City Hall Tourism Office

Profile Page
- Manage account information: photo, name, email,
  gender, phone number, password, status
- View booking history and receipts
- Check booking status (Confirmed, Pending, etc.)
- Modify settings or log out

Account Deletion
- Built-in account deletion feature for data privacy
- Accessible via Profile > Settings > Privacy Security
- Immediate and permanent removal of personal data,
  booking history, and credentials
- Support available via app or email:
  marikinatourismapp@gmail.com

--------------------------------------------------------
4. Admin Dashboard - Web  (Back-End)
--------------------------------------------------------
User Roles
- Admin: access to Users and Analytics pages
- Moderators: access to Bookings page
- Super Admin: full access to all features

Users Page
- Manage accounts and bookings
- Tabs: Users & Bookings, Notifications, Support Reports
- Editing, filtering, and refreshing tools

Bookings Page (CMS)
- Manage tour packages, destinations, and comments
- Edit or add packages and destinations
- Moderate user feedback

Analytics Page
- Visual insights and downloadable reports
- Metrics: bookings per day, booking mode,
  demographics, agency type distribution,
  revenue summary, package popularity,
  booking status, agency leaderboard,
  most viewed destinations
